Jailbreaking is the process of bypassing the security measures on a device (in this case an iPhone or iPad) in order to execute third-party code.

The process does come with a few security drawbacks, however the iOS jailbreak community is usually very good at mitigating these issues.

Jailbreaks are created using security vulnerabilities found in-the-wild by iOS security researchers, which are then exploited to gain root access on a device.
