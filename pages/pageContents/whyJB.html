Jailbreaking has many benefits.

Most people jailbreak so they can customize their device beyond the limits set for them by Apple.

Jailbreaking also allows for the installation of new system utilities, making an iPhone or iPad more usable for power users.

Got a lot of bulky system files and logs you don't need? Want custom message bubble shapes and colors? Want a custom home screen layout? Jailbreaking has you covered.

For more info on what you can do with a jailbreak, check out some setups on r/iOSThemes.
